Python

Comment syntax

有两种：

"""

或者

Build_in_function

内建函数指的是 __func__ 这样的函数。

以区别于：__func 以及 _func 。

关于内建函数可以参考：http://stackoverflow.com/questions/8689964/python-why-do-some-functions-have-underscores-before-and-after-the-functio

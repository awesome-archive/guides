使用 Jekyll 在 Github 上建立个人网站 (下)

通过本教程，我们想要学会：

  1. 如何在本地搭建一个可以运行 jekyll 的环境
  2. 如何使用 Github Page 来 Host 我们的网站。

接下来要做的工作

接下来在 PART3 我们会学习如何把本地的网站同步到 github 上并访问到。

PART 3：将网站同步到 Github 上，参考材料：

  • 如何使用 github page 搭建博客
  • Github Pages 使用入门
  • 博客搭建详细教程

Git的使用指南

  1. 使用 git 搭建自己的项目
  2. 首先在 github 新建一个仓库。（具体方法不详细叙述）
  3. 开启 github page 的服务。（登陆自己的 github 账号，打开刚刚新建过的仓库，然后点击设置(Settings) -> Github Pages -> Source -> Master Branch，

如何自定义域名：

  1. 登陆自己的 github 账号，打开需要设定域名的仓库，然后点击设置(Settings) -> Github Pages -> Custom domain，

如何修改网页内容：

如果使用模板的话，直接修改 Readme.md 的内容就行了，readme.md 的格式是 markdown 格式，markdown 的语法参考如下： - https://github.com/younghz/Markdown - http://wow.kuapp.com/markdown/

如果不想使用模板，就需要自学 html & css，可以参考：

  • https://www.zhihu.com/question/19661369 有哪些好的 HTML 和 CSS 入门教程可以推荐给新手？
  • codeacademy ：网站设计成一个一个的小 quiz，所以可以一边做一边学，
  • 之后将 readme.md 改成 index.html ，就可以显示自己开发的网页了。

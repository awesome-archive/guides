Ruby 里的 Hash（Dictionary）

# you can do hash like this
grades = {"A" => 100, "B" => 90}

# or this
options = {:size => 10, :font-family => "arial"}

# or this
options = {size:10, font-family:"arial"}

# can be asceess as 
options[size] # 10

# A hash can also be created by new method
grades = Hash.new

if you want to add default value then you can use

grades = Hash.new(0)

or by using default= method

grades.default = 0

更进阶的使用请参考

  • http://docs.ruby-lang.org/en/2.0.0/Hash.html

Notes

  • scope 在 C 里面，'{}' 的内部可以看到外部的变量。但是在 ruby 里面，scope 之间是严格划分开的，参见下面的例子，local_variables 之间是严格分开的。

    • scope.rb:
v1 = 1

class myClass
  v2 = 2 
  local_variables # => [:v2]
  def my_method
    v3 = 3
    local_variables
  end
end

obj = Myclass.new
obj.my_method # => [:v3]
local_variables # => [:v1]

TIPS: 在《Meteprogramming Ruby》一书里面，有一个 Bug 就是这个机制造成的。

wildflame's 完全自杀手册

Save The World

欢迎来到这里 - 网上最全面的中文自杀资源之一。

您可能正在阅读此手册，寻找有关自杀方法的相关信息。

在这里，你也许只是想知道有关自杀的统计信息，也许你感觉真的很糟，你想要结束它，你只是想要快乐。其实你并不是想要结束自己的生命，你只是想要终结痛苦。

我在 2018 年，就开始试图在网上搜索找到杀死自己的方法。然而我发现网上的信息少之又少，少的让我感到不知所措; 全部网站几乎都是治疗精神疾病和抑郁症的信息，以至于让我很难找到，我真正想要的信息; 而且那些防自杀热线没有任何卵用，他们不会给我钱，也不愿意帮我治疗我的疾病，甚至不会给我一份谋生的工作。

本网站/我并不是鼓励你自杀，但是同时我也不会限制你的自身选择，在这个压抑的时代（专制、物化），我们连安乐死的权利都没有。网站没有任何政治立场，也没有提供任何建议，这里没有公告板，聊天或论坛（虽然链接到一些）。我希望它是一种公正而且客观的资源，可以帮助您，并做出适合您的任何选择。

我希望你能发现它的用途。

在浏览网站之前，我希望你已经年满 18 岁，并且能够做出理性且正确的选择。

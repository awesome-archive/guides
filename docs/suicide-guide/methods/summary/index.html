选择合适的方法

概述

拥有一个关于自杀方法的信息库，包括各种方法的危险性及其可靠性，以及哪些方法最成功的统计信息。

在研究这个网站时，我对自杀方法写了多少感到震惊。许多公告板长期讨论不同的方法（通常是那些尝试过各种中毒和失败方法的人），书籍和网站。我发现有很多讨论很奇怪，面对很容易获得最自信的自杀方式的统计数据（参见最致命的自杀方法和自杀统计数据）。

圣杯似乎是无痛的自杀方法，人们将竭尽全力找到一种可能实现这一目标的方法。不幸的是，大量的这些尝试最终都失败了，因为干净，相对无痛的方法通常不是以这样的方式完成的，以致使它们致命。而且这些人中的许多人最终都在医院接受治疗失败。

自杀统计部分中的数字表明，每次成功的自杀企图都会有33个不成功的自杀企图。对于药物过量，这个比例大约是40比1.事实上，如果试图自杀，你最终有可能最终住院，无论是短期还是长期的健康影响，都比死亡还要大。

因此，如果您正在阅读这个更担心找到一个无痛方法而不是一个有效的方法，建议您仔细阅读本网站上提到的自杀方法的危险信息，然后或许看看帮助我，因为通过自杀而不是成功杀死自己，你更容易伤害自己。

对于任何致力于自杀的人来说，如果选择可靠的方法，实现目标可以是直截了当的。主要问题往往不是物流，而是内部自我分析，以及其他人如何反应的想法（我们的行为受到我们认为别人会如何反应的影响）。对于许多考虑自杀的人来说，他们有一种真正渴望结束自己生命的愿望，但他们害怕死亡可能会让他们无法做到这一点。

要了解有关个别自杀方法的更多信息，请从左侧菜单中选择。是的，那里没有提到的方法。在几乎所有情况下，因为它们要么不可靠，要么难以执行。因此，应该警告那些考虑不寻常方法的人 - 有一个原因，他们不是写得太多。
